## 🚀 Featured Projects (Portfolio)

Here are some of the main projects I have built during my studies at SMK Telkom Malang:

### 🎓 [EduAkses](https://github.com/4RBTR/EduAkses)
A unified platform designed to overcome fragmentation in digital education, combining classroom management, online meetings, and learning needs into one modern ecosystem.
* **Tech:** TypeScript, Next.js (Frontend)

### 💧 [PDAM-PROJECT (HydroFlow Systems)](https://github.com/4RBTR/PDAM-PROJECT)
A modern management platform for Perusahaan Daerah Air Minum (PDAM) designed to accelerate administrative processes, billing, and transaction reporting efficiently.
* **Tech:** TypeScript, Next.js / Backend Stack

### 🛒 [KlikBelanja](https://github.com/4RBTR/KlikBelanja)
A modern e-commerce platform built with Next.js 15 and Tailwind CSS. Designed with professional standards using modular architecture and data visualization.
* **Tech:** TypeScript, Next.js 15, Tailwind CSS

### ✈️ [HorizonAir](https://github.com/4RBTR/HorizonAir)
An executive-class modern flight ticket booking web application developed as a Project UKL (Ujian Kompetensi Keahlian) at SMK Telkom Malang.
* **Tech:** TypeScript, Full-stack Web Architecture

### 🏎️ [Prime-Wheels](https://github.com/4RBTR/Prime-Wheels)
A modern digital ecosystem for luxury and executive car rentals combining visual luxury with a high-level operational system.
* **Tech:** TypeScript

### 🔧 [Zippy-Rides](https://github.com/4RBTR/Zippy-Rides)
A full-stack web application for tracking vehicle maintenance, spare parts, service costs, and taxes—for both motorcycles and cars—in a single responsive dashboard.
* **Tech:** TypeScript, Full-stack
